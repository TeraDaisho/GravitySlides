# Gravity Slides (Japanese)

**Gravity Slides** は、ブラウザで表示しているスライドやWebページをキャプチャし、**Gemini AI** の力を使って自動的に洗練されたスピーカーノート付きのGoogleスライドを作成するChrome拡張機能です。

あなたのプレゼンテーション作成を、重力から解放されるようにスムーズにします。

## ✨ 機能

*   **ワンクリック・キャプチャ**: ブラウザ上のあらゆる画面（Canva、Figma、SpeakerDeckなど）を画像として取り込みます。
*   **AIスピーチ生成**: Gemini AIが、箇条書きのメモや音声入力テキストを、自然で説得力のある「話し言葉（スピーチ原稿）」に自動整形します。
*   **音声入力**: マイクを使って、スライドのアイデアやメモを声で入力できます。
*   **Googleスライド・エクスポート**: キャプチャした画像と生成された原稿を、ワンクリックでGoogleスライドに書き出します。
*   **スライド内テキスト表示**: スピーカーノートの内容をスライド上にテキストボックスとして配置することも可能。動画の字幕やカンペとして活用できます。
*   **セキュアな設計**: APIキーはGoogle Apps Script側で安全に管理されます。
*   **カスタマイズ**: 使用するGeminiモデルの選択や、プロンプトのカスタマイズが可能です。

## 🛠 導入方法

このツールは「Chrome拡張機能」と「Google Apps Script (GAS)」の2つの部分で構成されています。

### 手順1: Google Apps Script (GAS) のセットアップ

1.  [Google Drive](https://drive.google.com/) を開き、新しい **Google Apps Script** プロジェクトを作成します。
2.  エディタが開いたら、このリポジトリの `コード.gs` の中身をすべてコピーして貼り付けます。
3.  **スクリプトプロパティの設定**:
    *   左側の「設定（歯車アイコン）」を開きます。
    *   「スクリプトプロパティ」の「スクリプトプロパティを追加」をクリックします。
    *   **プロパティ**: `GEMINI_API_KEY`
    *   **値**: あなたの [Google AI Studio](https://aistudio.google.com/) で取得したAPIキーを入力して保存します。
4.  **権限の承認 (初回のみ)**:
    *   エディタ上の関数ドロップダウン（「デバッグ」の横）から **`authorizeScript`** を選択し、「実行」をクリックします。
    *   「権限を確認」というポップアップが出るので、Googleアカウントでログインし、**「詳細」>「安全ではないページに移動」>「許可」** をクリックして権限を付与します。
    *   「実行ログ」に「Authorization successful!」と表示されればOKです。
5.  **デプロイ**:
    *   右上の「デプロイ」>「新しいデプロイ」をクリックします。
    *   「種類の選択」で **「ウェブアプリ」** を選択します。
    *   **説明**: (例: v1)
    *   **次のユーザーとして実行**: **「自分」**
    *   **アクセスできるユーザー**: **「全員」** (これを選択しないと拡張機能からアクセスできません)
    *   「デプロイ」をクリックし、発行された **ウェブアプリのURL** をコピーしておきます。

### 手順2: Chrome拡張機能のインストール

1.  このリポジトリをダウンロード（ZIP解凍）またはクローンします。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の **「デベロッパーモード」** をオンにします。
4.  **「パッケージ化されていない拡張機能を読み込む」** をクリックし、ダウンロードしたフォルダ（`manifest.json` があるフォルダ）を選択します。

### 手順3: 初期設定

1.  Chromeのツールバーから **Gravity Slides** のアイコンをクリックしてサイドパネルを開きます。
2.  右上の **設定ボタン (⚙️)** をクリックします。
3.  **「GAS Webアプリ URL」** の欄に、手順1でコピーしたURLを貼り付けます。
4.  「保存して閉じる」をクリックします。

## 📖 使い方

1.  **スライドのキャプチャ**:
    *   ブラウザでスライド（Canvaのプレゼンテーションモードなど）を表示します。
    *   **「📸 スライドを撮影」** ボタンをクリックすると、現在の画面がリストに追加されます。
    *   右上の **「⤢ 16:9」** ボタンを押すと、ウィンドウサイズをYouTube動画などに最適な比率（1280x720など）にリサイズできます。

2.  **ノートの入力**:
    *   キャプチャしたスライドの下にあるテキストエリアに、説明したい内容をメモ書きします。
    *   **「🎤 音声入力を開始」** ボタンで、声をそのままテキスト化することも可能です。

3.  **書き出し**:
    *   すべてのスライドを撮り終えたら、最下部の **「🚀 スライドへ書き出し」** をクリックします。
    *   **「✨ Geminiで文章を整形」** にチェックを入れておくと、箇条書きのメモをAIが流暢なスピーチ原稿に書き換えてくれます。
    *   **「📝 スライド内にテキストを表示」** にチェックを入れると、スライドの左半分に半透明のテキストボックスを作成し、原稿を表示させることができます。
        *   ※このテキストボックスは、**Googleスライド上で自由に編集・移動・削除が可能**です。動画作成時のカンペや字幕として便利です。
    *   処理が完了すると完了メッセージとリンクが表示されます。

## ⚙️ 高度な設定

設定ボタン (⚙️) から以下のことができます。

*   **モデルの選択**: 使用するGeminiモデル（`gemini-1.5-flash`など）を変更できます。更新ボタンで最新モデルを取得できます。
*   **テキストボックスの透明度**: 「スライド内テキスト表示」機能を使う際の、テキストボックスの背景透明度をスライダーで0%〜100%まで調整できます。
*   **カスタムプロンプト**: AIへの指示を自由に書き換えられます。デフォルトでは日本語のスピーチ原稿用の指示が入っていますが、「英語に翻訳して」「関西弁にして」など自由に変更可能です。困ったら「↺ 初期化」ボタンで元に戻せます。

## 💡 Googleスライドで動画を作成するヒント

Gravity Slidesで作ったスライドを使えば、簡単に解説動画を作成できます。

1.  **書き出されたスライドを開く**: 内容を確認し、微調整を行います。
2.  **スピーカーノートを読む**: AIが作成した原稿は、そのまま読み上げるだけで自然な解説になるよう設計されています。
3.  **画面録画**:
    *   Googleスライドの「スライドショー」>「プレゼンター表示」を使います。
    *   OBS StudioやLoomなどの録画ツールで、スライド画面だけを録画範囲に指定します。
    *   自分はスピーカーノートを見ながら、ページをめくって読み上げていきます。
4.  **音声合成ソフトを使う場合**:
    *   スピーカーノートのテキストをコピーし、VOICEVOXなどの音声合成ソフトに貼り付ければ、AI音声による解説動画もすぐに作れます。
